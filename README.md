# Techfest 2025 | Campus Ambassador | IIT Bombay

A futuristic cyberpunk-themed landing page for Techfest IIT Bombay's Campus Ambassador Program.

## Features
- 🎨 Cyborg/Neon aesthetic design
- ✨ Interactive animations & particle effects
- ⌨️ Custom cursor with glow effects
- 📱 Fully responsive layout
- ⏰ Live countdown to Techfest 2025
- 🖱️ Intersection Observer animations

## Tech Stack
- HTML5
- CSS3 (Animations, Grid, Flexbox)
- JavaScript
